
## 🚀 How to Run the Application and Tests

### 1. Install dependencies

```bash
yarn install
```

### 2. Start the React application locally

```bash
yarn start
```

### 3. Run Cypress tests
a) Run tests headlessly

```bash
yarn test
```

b) Open Cypress Test Runner (interactive mode)

```bash
yarn cypress:open
```

### ⚠️ Notes

- Always start the app (yarn start) before running Cypress tests locally to ensure the application is available.

- Test reports and screenshots are automatically generated by Cypress and can be found inside the cypress folder after tests run.
